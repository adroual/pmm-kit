# Linked Projects

## How to Use

1. Run `/pmm.link <path-or-url>` to link a feature project (local path or Notion URL)
2. Run `/pmm.link` with no argument to pick CommDocs from Notion interactively
3. Run `/pmm.sync` to pull latest content from linked projects
4. Run `/pmm.narrative` to generate the unified narrative
