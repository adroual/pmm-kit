# {{project.name}} — PMM Workspace

This folder is managed by **pmm-kit**.

## How to use it

1. Open this project in your AI agent (Claude Code, Gemini CLI, Cursor, etc.).
2. Run:
   - `/pmm.constitution` to set up brand voice & PMM rules.
   - `/pmm.research` to turn raw notes into a research dossier.
   - `/pmm.commdoc` to generate the CommDoc from `input/`.
   - `/pmm.gtm` to derive a GTM plan from the CommDoc.
   - `/pmm.narrative` to build the narrative playbook.
   - `/pmm.sales-playbook` for Sales talk tracks.
   - `/pmm.sales-enablement` for the Sales enablement summary.
   - `/pmm.success-report` at the end of the launch.
   - `/pmm.changelog` when you need a public-facing changelog.
3. Edit the generated Markdown files as needed and commit to git.
